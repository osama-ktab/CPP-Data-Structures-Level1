# C++ Data Structures (Level 1) 🚀

Welcome to my Data Structures repository! This repository documents my practical implementation and deep dive into foundational **Data Structures** using C++. The primary focus here was transitioning from static memory containers to dynamic, custom-built data structures, ensuring optimized memory usage and efficient data manipulation.

## 📌 Repository Core Objectives
* **Custom Implementations:** Building and simulating core data structures from scratch to understand pointers, nodes, and references.
* **Dynamic Memory Management:** Mastering the utilization of heap memory, avoiding memory leaks, and understanding pointer manipulation.
* **Algorithmic Efficiency:** Analyzing and implementing operations like insertion, deletion, searching, and traversal within dynamic structures.
* **Standard Template Library (STL):** Getting hands-on experience with C++ built-in containers and comparing them with custom-built structures.

---

## 🛠️ Key Structures & Concepts Implemented
* **Singly Linked List:** Node creation, head/tail insertions, deleting nodes, reversing lists, and complex pointer mapping.
* **Doubly Linked List:** Managing bi-directional node pointers, forward/backward traversals, and mid-list manipulation.
* **Dynamic Arrays & Vectors:** Simulating size-doubling logic, deep copying, and optimizing sequential storage.
* **Stacks & Queues:** Applying LIFO (Last In, First Out) and FIFO (First In, First Out) principles using both arrays and linked lists.

---

## 📂 Project Structure
The repository is structured to showcase standard engineering modularity. Each folder or file contains:
1. **Header/Class Definitions:** Pure architectural blueprints for the data structure.
2. **Implementation & Test Driver:** A main execution file testing edge cases (e.g., deleting from an empty list, out-of-bounds access).
